# Environment Setup
Environment Setup
1. If running with Docker, copy the .env.example file to .env:
   ```bash
   cp .env.example .env
   ```
2. If running locally, copy the .env.example file to .env and add your environment parameters:
   ```bash
   cp .env.example .env
   ```
   Edit the .env file using any text editor

# Setup
Run "setup/setup.sh" from frontend root folder:
```bash
bash setup/setup.sh
```

# Usage
Run "python3 LibApp.py" from frontend root folder
```bash
python3 LibApp.py
```

# Tests 
Run "bash test/unit/run_tests_cov.sh" from frontend root folder
```bash
bash test/unit/run_tests_cov.sh
```
